# HTML_Portfolio
First time creating a portfolio website

A simple resume website built using **only HTML** as part of the HTML Capstone Project from Angela Yu's Web Development course.

## 📖 About the Project

This project is a personal resume website created using pure HTML without any CSS or JavaScript. The objective was to strengthen my understanding of HTML fundamentals by building a real-world project.

The website presents my education, skills, projects, certifications, and other professional information in a clean and well-structured format.

## 🎯 Objectives

* Practice core HTML concepts
* Learn how to structure a complete web page
* Use semantic HTML elements
* Build a project suitable for a GitHub portfolio
* Gain hands-on experience before learning CSS

## 🛠️ Built With

* HTML5
